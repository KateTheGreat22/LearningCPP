# LearningCPlus
# Overview

Within this C++ project, I was learning how to grapple the different elements within C++. As I have used other C languages over the course of my undergraduate degree, this assignment helped me to become comfortable and confident with the specific delimiters and notations used for C++. Please note that I used AI towards the end of the assignment to help me troubleshoot getting the pieces to align correctly.

This code walks the user through a simple game of chess. It can be played with two people as the program will switch between the users in the same terminal. The purpose in writing this game was to improve my ablities to make and use different classes and use them across several different files.
